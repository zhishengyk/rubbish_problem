## 题目描述
在米仓中有 $N$ 粒米，每粒米代表一个数字。给定一个数字 $C$，需要找出所有满足 $A - B = C$ 的数对 $(A, B)$ 的个数。注意，即使数字相同但位置不同也算作不同的数对。

## 输入格式
- 第一行两个整数 $N$, $C$；
- 第二行 $N$ 个整数，表示米仓中的数字序列。

## 输出格式
输出满足 $A - B = C$ 的数对个数。

## 样例

### 输入
```
4 1
1 1 2 3
```

### 输出
```
3
```

## 数据范围与提示
- $0 \le N \le 2 \times 10^5$；
- 所有数据小于 $2^{30}$。
