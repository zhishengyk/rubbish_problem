
## 题目描述
给定一个十进制正整数 $n$ ($0<n<1000000000$)，每个数位上数字均不为 $0$。$n$ 的位数为 $m$。现在从 $m$ 位中删除 $k$ 位 ($0<k<m$)，求生成的新整数最小为多少？

## 输入格式
- 第一行 $t$，表示有 $t$ 组数据
- 接下来 $t$ 行，每一行表示一组测试数据，每组测试数据包含两个数字 $n$ 和 $k$

## 输出格式
$t$ 行，每行一个数字，表示从 $n$ 中删除 $k$ 位后得到的最小整数

## 样例

### 输入样例1
```
2
9128456 2
1444 3
```

### 输出样例1
```
12456
1
```
