
## 题目描述
将给定的十进制数n转换为以-R为基数的负进制数表示（-20 ≤ R ≤ -2，|n| ≤ 37336）。当基数超过10时，使用字母表示大于9的数字。

## 输入格式
每行两个输入数据：
- 十进制数n
- 负进制数的基数-R

## 输出格式
转换后的负进制数及其基数，格式为n=result(base-R)

## 样例
### 输入#1
```
30000 -2
```
### 输出#1
```
30000=11011010101110000(base-2)
```

### 输入#2
```
28800 -16
```
### 输出#2
```
28800=19180(base-16)
```
