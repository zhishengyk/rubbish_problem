
## 题目描述
津津、菲菲和皮皮参加哈尔滨的冰灯艺术展。他们发现每一盏冰灯都有一个独特的编号，这些编号都是正整数，构成了1个正整数数列。现在牛老师要求3人在这个数列中找出所有编号之差为$C$（$A-B=C$）的数对。注意，不同位置的编号即使相同，也算作不同的数对。

## 输入格式
输入共两行。
第一行，两个整数$N$, $C$
第二行，$N$个整数，作为要求处理的那串数。

## 输出格式
一行，表示该串数中包含的满足$A-B=C$的数对的个数。

## 样例

### 输入样例
```
4 1
1 1 2 3
```

### 输出样例
```
3
```

## 数据范围与提示
$0 \leq n \leq 2 \times 10^5$
所有数据小于$2^{30}$
