## 题目描述
在文昌公园里有 $R$ 个水龙头供游客们使用。假设某一天 $N$ 个游客依次前来打水，他们装满各自水杯所需的时间分别是 $T_1, T_2, ..., T_n$，其中每个时间 $T_i$ 都是整数且各不相同。牛老师要求津津、菲菲和皮皮3人写一个程序，优化游客们打水的顺序，使得他们总共花费的时间最少。

## 输入格式
第1行只有一个整数 $n$ 和 $r$，第2行有 $n$ 个正整数，是每个人的打水时间。相邻两个整数之间用单个空格隔开。

## 输出格式
输出总共打水时间。

## 样例

### 输入样例1
```
4 2
2 6 4 5
```

### 输出样例1
```
23
```

## 数据范围与提示
- $1 \leq n \leq 500$
- $1 \leq r \leq 100$
- $1 \leq T_i \leq 100$
