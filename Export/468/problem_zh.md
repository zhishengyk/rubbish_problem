## 题目描述
我军拦截敌军传递了2个数字串，后经侦查得知，这两个数字串中包含了一个很重要的密码，这个密码的破译方式为：
1. 用这两个数字串中的大数 - 小数
2. 将得到的差值转换为16进制（使用大写字母）

## 输入格式
输入有2行，分别是2个不超过200位的整数，且已知2个整数的差是一个不超过18位的整数。

## 输出格式
输出按题意计算出的16进制数。

## 样例
### 输入样例1
```
99999999999999999999999999999999999999973
99999999999999999999999999999999999999999
```

### 输出样例1
```
1A
```

### 输入样例2
```
999999999
3847
```

### 输出样例2
```
3B9ABAF8
```
