## 问题192 题解

### 题目分析

这是经典的“百钱买百鸡”问题的一个泛化版本。我们需要求解一个三元一次不定方程组：
- `5x + 3y + z/3 = n`
- `x + y + z = m`

### 解题思路

这个问题可以通过枚举法解决，但可以通过消元来优化。

1.  **消元**：
    -   将两个方程联立，消去变量 `z`，得到 `7x + 4y = (3n - m) / 2`。

2.  **枚举**：
    -   我们可以枚举公鸡的数量 `x`。
    -   对于每个枚举的 `x`，我们可以计算出 `4y = (3n - m) / 2 - 7x`。
    -   检查 `( (3n - m) / 2 - 7x )` 是否能被4整除，如果可以，就计算出 `y`。
    -   然后，我们再计算 `z = m - x - y`，并验证 `y >= 0` 和 `z >= 0` 是否都成立。
