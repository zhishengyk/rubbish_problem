## 题目描述
津津、菲菲和皮皮准备去吉林市著名景区松花湖去玩，他们有一辆小车，可是这辆小车除了菲菲开车外只能带1人，已知津津、皮皮两人的步行速度一样，且小于车的速度。问：怎样利用小车才能使两人尽快同时到达。

## 输入格式
仅一行，三个实数，分别表示AB两地的距离$s$，人的步行速度$a$，车的速度$b$。

## 输出格式
两人同时到达B地需要的最短时间，保留6位小数。

## 样例

### 输入样例
```
120 5 25
```

### 输出样例
```
9.600000
```

## 数据范围与提示
对于$100\%$的数据，保证$0 \leq s, a, b \leq 10^9$
