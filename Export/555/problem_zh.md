
## 题目描述

鄂尔多斯的羊毛衫享誉全世界，特别是 AI 全自动针织技术更是领先全球，津津、菲菲和皮皮都想体验一下。已知织机上有 N 个不同颜色毛线球（用不同数字表示）组成的毛线链，它会根据设计图纸进行 M 次操作最终完成产品。

他们了解到织机的每次操作可以是以下 2 种类型之一：
1. 操作 1：在第 X 个毛线球之后插入一个新的毛线球，其颜色为 Y。
2. 操作 2：删除第 X 个毛线球。

3 人决定用程序模拟这个过程，并输出最终毛线链的颜色序列。

## 输入格式

第一行两个整数 N，M（N，M ≤ 100000），N 表示数组中一开始有 N 个数，M 表示 M 次操作。

第二行 N 个整数，表示原来的数组。

接下来 M 行，每行第一个数 OPT，表示操作类型：
- 对于操作 1，接下来两个数 X，Y，表示在第 X 数之后插入一个数 Y，保证 0 ≤ X ≤ 当前数的个数，若 X=0，表示在数组开头插入。
- 对于操作 2，接下来一个数 X，表示要删除第 X 个数，保证 1 ≤ X ≤ 当前数的个数。

## 输出格式

输出若干个数，表示最后的数组。

## 样例

### 输入样例 #1
```
5 3
1 2 3 4 5
1 1 6
2 1
2 2
```

### 输出样例 #1
```
6 3 4 5
```
