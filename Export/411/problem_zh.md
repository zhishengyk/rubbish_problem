## 题目描述
香港大学根据笔试成绩划定面试分数线，面试分数线为计划录取人数m的150%（向下取整）名选手的分数。输出所有进入面试的选手信息。

## 输入格式
第一行两个整数n，m（5≤n≤5000，3≤m≤n），表示报名选手总数和计划录取人数。接下来n行，每行包含选手的报名号和笔试成绩。

## 输出格式
第一行输出面试分数线和实际进入面试人数。之后按成绩从高到低输出进入面试的选手信息，成绩相同按报名号升序排列。

## 样例
### 输入 #1
```
6 3
1000 90
3239 88
2390 95
7231 84
1005 95
1001 88
```

### 输出 #1
```
88 5
1005 95
2390 95
1000 90
1001 88
3239 88
```
