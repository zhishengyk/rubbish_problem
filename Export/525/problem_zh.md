## 题目描述
给定一个长度为$n$的数组$a_1,a_2,\ldots,a_n$。请你找到其中的最佳连续子数组。最佳连续子数组需满足：
1. 子数组内各元素的算术平均数（即所有元素之和除以元素个数）尽可能大
2. 满足条件1的前提下，子数组的长度尽可能长

输出最佳连续子数组的长度。

## 输入格式
第一行包含整数$n$。
第二行包含$n$个整数$a_1,a_2,\ldots,a_n$。

## 输出格式
每组数据输出一行结果，表示最佳连续子数组的长度。

## 样例

### 输入样例
```
5
6 1 6 6 0
```

### 输出样例
```
2
```

## 数据范围与提示
$1 \leq n \leq 10^5$
