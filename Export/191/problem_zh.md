
## 题目描述

一天，小雨在仓库里找到了一个序列 $A$，$N$ 是它的长度。小雨希望序列的所有元素**全部相等**，可以进行以下操作：

- 选择一个下标 $i$，把 $A_i$ 变为 $(A_i+1)\%10$

现在给定序列，求最少需要操作多少次使所有元素相等。

## 输入格式

第一行一个正整数 $N$。  
接下来 $N$ 行是序列元素 $A_1, A_2, \ldots, A_N$。

## 输出格式

输出使所有元素相等的最少操作次数。

## 样例

### 输入1
```
3
1 2 3
```

### 输出1
```
3
```

### 输入2
```
4
1 1 1 1
```

### 输出2
```
0
```

## 数据范围
- 对于60%的数据：$1 \le N \le 10^3$，$0 \le A_i \le N$
- 对于100%的数据：$1 \le N \le 10^5$，$0 \le A_i \le N$

## 样例解释
- 样例1：将1→2→3（2步），2→3（1步），共3步
